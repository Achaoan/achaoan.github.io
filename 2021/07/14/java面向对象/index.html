java面向对象

本文最后更新于：31 分钟前

一、面向对象的三大特性

1. 封装（Encapsulation）

把数据和操作数据的方法绑定起来，对数据的访问只能通过已定义的接口。通过访问权限修饰符控制访问级别。

目的：隐藏实现细节，对外提供简单的接口，提高安全性、可维护性。

2. 继承（Inheritance）

子类继承父类的属性和方法，实现代码复用。Java 只支持 单继承（一个类只能有一个直接父类），但可以通过接口实现多继承。

注意：优先使用组合而非继承（Composition over Inheritance），因为继承破坏封装性，父类变更可能影响所有子类。

3. 多态（Polymorphism）

同一方法在不同对象上表现出不同行为。分为两类：

  • 编译时多态：方法重载（Overload）
  • 运行时多态：方法重写（Override）+ 父类引用指向子类对象（向上转型）

三个必要条件：

  1. 继承（或接口实现）
  2. 方法重写
  3. 父类引用指向子类对象

多态的底层是 动态绑定（运行时确定实际类型的方法），通过虚方法表（vtable）实现。

三、常见面试题

1. 重写（Override）和重载（Overload）的区别？

维度 重载（Overload） 重写（Override）
发生位置 同一个类中 子类中
参数列表 必须不同 必须相同
返回类型 可以不同 必须相同（或协变返回类型）
权限修饰符 不限 不能更严格
绑定时机 编译期（静态多态） 运行期（动态多态）

2. 抽象类和接口的区别？

维度 抽象类 接口
构造方法 有 无
成员变量 任意类型 public static final 常量
方法 抽象方法 + 普通方法 抽象方法 + default/static 方法（Java 8+）
继承方式 单继承 多实现
设计思想 is-a 关系，代码复用 like-a 关系，行为契约

3. 构造方法可以被重写吗？

不能。构造方法名必须与类名相同，子类构造方法名与子类类名相同，所以不存在重写概念。但构造方法可以重载。

4. equals() 和 == 的区别？

比较方式 基本数据类型 引用数据类型
== 比较值是否相等 比较内存地址是否相同
equals()（未重写） 不适用 等同于 ==
equals()（已重写，如 String） 不适用 比较内容是否相等

重要约定：重写 equals() 必须重写 hashCode()。

5. 静态方法能被重写吗？

不能。静态方法属于类，不属于实例，没有多态性。在子类中定义同名静态方法称为 隐藏（Hide），而非重写。

6. 浅拷贝和深拷贝的区别？

  • 浅拷贝：基本类型传值，引用类型传引用（新旧对象共享引用对象）
  • 深拷贝：基本类型传值，引用类型创建新对象
