六、面试题

1. 雪花算法的 ID 结构？

1 位符号 + 41 位时间戳 + 10 位机器 ID + 12 位序列号，共 64 位 long。

2. 时钟回拨怎么处理？

小幅度（≤ 5ms）等待追赶，大幅度抛异常。或用 Redis/美团 Leaf 方案不依赖本地时钟。

3. 机器 ID 怎么分配？

配置文件、ZK 协调、Redis 自增、IP 取模。

4. 雪花算法和 UUID 对比？

雪花算法 64 位整型（8 字节），趋势递增，适合做数据库主键。UUID 128 位（36 字节），无序，不适合 B+ 树。

5. 美团 Leaf 的号段模式怎么优化？

双 buffer 异步加载，buffer 1 快用完时后台预加载 buffer 2，消除 DB 访问毛刺。
