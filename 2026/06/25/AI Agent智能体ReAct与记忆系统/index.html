AI Agent 智能体：ReAct 模式、记忆系统与规划策略

本文最后更新于：22 分钟前

一、什么是 AI Agent

AI Agent（智能体）是能够自主感知环境、做出决策、执行行动的 LLM 驱动系统。

二、ReAct 模式

ReAct（Reasoning + Acting）是最主流的 Agent 模式，让模型交替进行推理和行动。它的名字来自论文《ReAct: Synergizing Reasoning and Acting in Language Models》。

六、面试题

1. AI Agent 和普通 LLM 调用的区别？

普通 LLM 调用是一问一答无状态的。Agent 能感知环境、自主规划、调用工具、管理记忆，是目标驱动的多轮系统。Agent = LLM + 工具 + 记忆 + 规划。

2. ReAct 模式的三步循环是什么？

Thought（出声思考当前状态和下一步）→ Action（调用工具）→ Observation（工具返回结果）。循环直到任务完成。

3. Agent 的三种记忆？

短期记忆（当前对话，在上下文窗口中）、长期记忆（跨会话持久化，存在外部存储中）、工作记忆（当前任务进度）。通过 RAG 检索长期记忆回上下文。

4. Agent 的规划策略有哪些？

任务分解（拆成子步骤）、反思（定期检查和修正）、多路径搜索（探索多种方案）、子 Agent 协作（分工合作）。
