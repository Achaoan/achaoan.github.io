八、面试题

1. 如何查找日志中某个关键字的出现次数？

grep -c "keyword" app.log 或 grep "keyword" app.log | wc -l。

2. 如何查看 CPU 占用最高的进程？

top 按 P 键排序，或 ps aux --sort=-%cpu | head -5。

3. 如何找出磁盘空间占用最大的目录？

du -sh /* | sort -rh | head -10 或 ncdu。

4. nohup 和 & 的区别？

&：后台运行（关闭终端会退出）。nohup：忽略 HUP 信号（关闭终端不退出）。通常组合使用 nohup command &。

5. 网络排查常用命令有哪些？

ping（连通性）、telnet/nc（端口）、ss（连接）、curl（HTTP）、traceroute（路由追踪）、nslookup/dig（DNS）。

6. 如何查看某个 Java 进程的线程堆栈？

jstack <PID> 或 kill -3 <PID> 输出到标准输出。
