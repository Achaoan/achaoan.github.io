一、Netty 概述

什么是 Netty？

Netty 是一个异步事件驱动的网络应用框架，用于快速开发高可用的网络服务器和客户端。

八、面试题

1. Netty 的线程模型？

主从 Reactor 模型：BossGroup 处理 Accept，WorkerGroup 处理 IO 读写。

2. EventLoop 是什么？

一个 EventLoop = 一个线程 + 一个 Selector，处理多个 Channel 的 IO 事件。

3. Netty 的零拷贝怎么实现？

FileRegion（sendfile）、CompositeByteBuf（组合不拷贝）、wrappedBuffer（包装数组不拷贝）。

4. ChannelPipeline 是什么？

责任链模式的处理器链，Inbound 处理读，Outbound 处理写。

5. Netty 和 Tomcat 的区别？

Netty 是 NIO 异步框架，适合高并发长连接（如 WebSocket、RPC）。Tomcat 是 Servlet 容器，适合 HTTP 请求响应模式。
