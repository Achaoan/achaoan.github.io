七、面试题

1. LangGraph 的核心概念？

State（共享状态）、Node（处理步骤）、Edge（执行顺序）、Graph（节点+边的集合）。通过有向图建模 Agent 工作流，支持循环边实现 ReAct 模式。

2. AutoGen 的多 Agent 协作模式？

Supervisor-Worker（中心化分工）、Debate（辩论提升质量）、Pipeline（流水线处理）、Swarm（冗余容错）。通过 Agent 间的对话协作。

3. CrewAI 的特点？

将 Agent 定义为角色（Role + Goal + Backstory），组成”团队”按流程执行任务。比 AutoGen 控制性更强，适合明确分工的场景。

4. A2A 协议解决什么问题？

不同 Agent 框架之间的互操作。让 LangGraph、AutoGen、CrewAI 等不同框架的 Agent 可以互相通信和协作。MCP 是 AI→Tool，A2A 是 Agent→Agent。

5. 如何选择 Agent 框架？

LangGraph 适合复杂流程和生产环境，AutoGen 适合多 Agent 对话协作，CrewAI 适合角色分工明确的任务。快速原型选 Dify/Coze，Java 生态选 Spring AI。
