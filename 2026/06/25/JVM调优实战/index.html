七、面试题

1. JVM 调优常用参数有哪些？

-Xms/-Xmx（堆大小）、-Xss（栈大小）、-XX:+UseG1GC（GC 选择）、-XX:MaxGCPauseMillis（暂停目标）。

2. 如何排查 OOM？

堆转储（HeapDumpOnOutOfMemoryError）→ MAT 分析 → 找 GC Roots 引用链 → 定位代码问题。

3. G1 和 CMS 的区别？

CMS：老年代并行 GC，关注低延迟，有碎片问题。G1：分区管理，可预测暂停时间，JDK 9+ 默认。

4. 内存泄漏和内存溢出的区别？

内存泄漏（Memory Leak）：对象不再使用但 GC 无法回收，导致可用内存越来越少。内存溢出（OOM）：内存不足，无法分配新对象。

5. 如何确定堆大小？

先评估业务：活跃数据量 * (3-5 倍) = 堆大小。通过压测验证，观察 GC 频率（YGC 1-2 次/秒，Full GC 尽量不发生）。

6. jstack 能分析什么问题？

死锁、线程阻塞、CPU 高（找热点线程）、线程池耗尽、长时间等待等原因分析。
