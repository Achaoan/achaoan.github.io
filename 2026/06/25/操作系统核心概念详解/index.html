六、面试题

1. 进程和线程的区别？

进程是资源分配最小单位，有独立地址空间；线程是 CPU 调度最小单位，共享进程地址空间。线程切换比进程切换快。

2. 死锁的必要条件？

互斥、持有并等待、不可剥夺、循环等待。四个条件缺一不可。

3. 如何解决死锁？

预防（破坏四个条件之一）、避免（银行家算法）、检测恢复（资源分配图）、忽略（鸵鸟算法，实际最常用）。

4. 虚拟内存的作用？

隔离地址空间、部分加载、共享内存。通过页表将虚拟地址映射到物理地址。

5. LRU 如何实现？

硬件置位访问位，Linux 使用 active/inactive 链表管理。近似 LRU：每次访问标记访问位，淘汰时扫描清除。

6. 上下文切换时发生了什么？

保存当前进程的寄存器、程序计数器、页表 → 加载新进程的上下文 → TLB 刷新 → 开始执行新进程。
