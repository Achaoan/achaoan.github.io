六、面试题

1. CI/CD 是什么？

持续集成（自动构建测试），持续交付/部署（自动发布）。

2. GitLab CI 的基本概念？

.gitlab-ci.yml 定义流水线，GitLab Runner 执行 Job，多个 Stage 组成 Pipeline。

3. Jenkins Pipeline 的两种类型？

声明式（Declarative，结构清晰）和脚本式（Scripted，灵活但复杂）。推荐声明式。

4. Webhook 的作用？

代码提交时 Git 仓库通知 CI 工具触发构建，不需要手动启动。

5. 镜像构建怎么做？

多阶段构建减少镜像体积，Dockerfile 中先缓存依赖层，再编译代码层。
