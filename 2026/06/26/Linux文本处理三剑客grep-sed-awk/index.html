1. grep、sed、awk 各自擅长什么？

grep 按行查找匹配模式；sed 按行替换/删除/插入文本；awk 按列分析和统计结构化文本。grep 是”搜索”，sed 是”编辑”，awk 是”分析”。

2. 统计 access.log 中每个 IP 的访问次数？

awk '{ips[$1]++} END {for(ip in ips) print ip, ips[ip]}' access.log | sort -k2 -rn | head -10

3. sed ‘s/old/new/g’ 各部分的含义？

s 是替换命令，old 是匹配模式，new 是替换文本，g 是全局标志（同一行所有匹配都替换，不加 g 只替换第一个）。

4. 如何统计日志中某个关键字的出现次数？

grep -c "keyword" app.log 或 grep "keyword" app.log | wc -l 或 awk '/keyword/ {count++} END {print count}' app.log。
