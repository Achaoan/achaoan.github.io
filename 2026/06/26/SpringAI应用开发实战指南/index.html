九、面试题

1. Spring AI 是什么？解决了什么问题？

Spring AI 是 Java 生态的 AI 应用框架，统一了大模型调用 API，提供了 ChatClient、RAG、Tool Calling、Agent 等开箱即用的能力，解决 Java 接入 LLM 缺乏标准化方案的问题。

2. RAG 的原理和流程？

用户提问 → 向量化 → 向量数据库检索相似文档 → 文档作为上下文拼入 Prompt → LLM 基于上下文生成回答。解决 LLM 知识过时和幻觉问题。

3. Tool Calling 是什么？

让 LLM 在对话中调用外部 Java 方法（数据库查询、API 调用等），扩展 AI 能力边界。Spring AI 中通过 @Tool 注解实现。

4. AI Agent 的 ReAct 模式？

思考（Thought）→ 执行（Action）→ 观察（Observation）循环，直到得到最终答案。Agent 能自主规划、调用工具、处理多步任务。

5. 如何选择大模型？

国内生产推荐 DeepSeek（性价比高）、通义千问（阿里生态）；本地开发推荐 Ollama（免费离线）；海外推荐 OpenAI GPT-4o。
