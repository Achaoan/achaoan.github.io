七、面试题

1. 如何排查 CPU 占用高的 Java 问题？

四步法：top -H -p <PID> 找 CPU 高的线程 → printf "%x" 转 16 进制 → jstack 查看线程栈 → 分析代码。常用工具还有 Arthas、perf。

2. 如何查看磁盘 IO 压力？

iostat -x 1 查看 %util（磁盘忙碌率）和 await（响应时间）。%util > 80% 表示磁盘压力大，await > 10ms 表示响应慢。

3. 内存不足怎么排查？

free -h 查看整体使用，vmstat 1 查看 swap 换页，ps aux --sort=-%mem 找内存最高的进程。Java 用 jmap -histo 看对象分布。

4. 网络排查常用命令？

ping（连通性）、telnet/nc（端口）、ss（连接状态）、curl（HTTP 耗时分析）、traceroute/mtr（路由路径）、tcpdump（抓包分析）。

5. nohup 和 & 的区别？

& 后台运行，但关闭终端会收到 SIGHUP 信号退出。nohup 忽略 SIGHUP 信号，关闭终端不退出。通常组合使用 nohup command &。
