七、面试题

1. WebSocket 和 HTTP 的区别？

HTTP 是单向请求-响应模式，WebSocket 是双向全双工通信。WebSocket 建立后头部开销极小（2 字节 vs HTTP 头部几百字节）。

2. WebSocket 握手过程？

客户端发送 HTTP Upgrade 请求（含 Sec-WebSocket-Key），服务端计算 Sec-WebSocket-Accept 返回 101 状态码，之后升级为 WebSocket 协议。

3. WebSocket 支持二进制吗？

支持。WebSocket 帧有 opcode 字段：0x1 文本、0x2 二进制、0x8 关闭、0x9 Ping、0xA Pong。

4. Spring 中 @SendTo 和 convertAndSendToUser 的区别？

@SendTo 广播到所有订阅者，convertAndSendToUser 点对点发送（自动添加 /user/{username} 前缀）。

5. WebSocket 集群怎么处理？

使用 Redis Pub/Sub 或 MQ 广播消息：Server1 收到消息 → Redis 发布 → Server2 订阅后推送给本地 WebSocket 连接。
