六、面试题

1. ArrayList 和 LinkedList 的区别？

ArrayList 基于数组，随机访问 O(1)，尾部插入快，中间插入慢。LinkedList 基于链表，随机访问 O(n)，任意位置插入删除快。ArrayList 更节省内存（没有指针开销）。

2. HashMap 的 put 流程？

计算 hash → 定位桶 → 桶空直接插入，桶不空遍历链表/树 → key 存在覆盖，key 不存在插入 → 链表超长树化 → 元素超阈值扩容。

3. 栈和队列的应用场景？

栈：函数调用、括号匹配、表达式求值、撤销操作、DFS。
队列：BFS、生产者消费者、异步任务、滑动窗口、消息队列。

4. 链表快慢指针有什么用？

找中点（slow 走一步，fast 走两步）、检测环（相遇即有环）、找倒数第 K 个（fast 先走 K 步）、回文判断（找中点 + 反转后半）。
