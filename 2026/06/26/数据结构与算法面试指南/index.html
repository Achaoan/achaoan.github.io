六、面试题

1. ArrayList 和 LinkedList 的区别？

ArrayList 基于数组，随机访问 O(1)，插入删除 O(n)。LinkedList 基于链表，随机访问 O(n)，插入删除 O(1)。

2. HashMap 的底层实现？

数组 + 链表 + 红黑树。put 时计算 hash → 定位桶 → 插入（链表/树）。容量为 2 的幂，负载因子 0.75，扩容翻倍。

3. 排序算法中哪个最快？

实际场景中快排最快（时间复杂度 O(n log n)，空间复杂度 O(log n)，缓存友好）。但最坏 O(n²)，归并排序更稳定。

4. DFS 和 BFS 的区别？

DFS 用栈（递归或显式栈），适合路径搜索、拓扑排序。BFS 用队列，适合最短路径、层序遍历。

5. 动态规划的核心思想？

重叠子问题 + 最优子结构。将大问题分解为子问题，避免重复计算。关键是状态定义和状态转移方程。
