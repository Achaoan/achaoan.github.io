八、面试题

1. Nacos 和 Eureka 的区别？

Nacos 支持 AP+CP 混合模式，提供配置中心功能，有 Web 控制台。Eureka 只有注册中心功能，2.0 已停止开发。

2. Nacos 如何实现动态配置刷新？

@RefreshScope 注解 + Nacos 配置中心。配置变更时 Nacos 推送通知，应用重新绑定 @Value 属性。

3. OpenFeign 的原理？

运行时生成接口的代理对象，通过 @FeignClient 解析注解拼接 URL，使用 RestTemplate/HTTP Client 发起请求。

4. Gateway 和 Zuul 的区别？

Gateway 基于 WebFlux（Reactor），非阻塞异步，性能高。Zuul 1.x 基于 Servlet，同步阻塞。Gateway 支持 WebSocket。

5. 网关一般做什么？

统一鉴权、限流、路由转发、日志记录、跨域处理、协议转换（HTTP→Dubbo）。

6. Feign 调用超时怎么配置？

connect-timeout 连接超时，read-timeout 读取超时。可在全局或指定服务级别配置。同时需要配置 Ribbon/LoadBalancer 的超时。
